Я хочу сделать веб-приложение "Трекер привычек".
Стек: Next.js + Supabase + Tailwind — это будет наша связка.

Вот что мне нужно:
- список привычек, можно добавлять новые и отмечать выполнение за день
- графики прогресса за последнюю неделю для каждой привычки
- статистика: процент выполнения, лучшая привычка, текущий streak
- красивый дизайн — тёмная тема, анимации, glassmorphism карточки
- данные хранятся в Supabase, чтобы работало из любого браузера
- в конце — деплой на Vercel

## Авторизация

Нужна авторизация через Supabase Auth — регистрация и вход по email/password.
Неавторизованные пользователи перенаправляются на страницу логина.
У каждого пользователя свои привычки и свой прогресс.

Данные

Используем стандартных пользователей из Supabase Auth.
Помимо этого нужно хранить:

привычки пользователя (название, описание, цвет)
логи выполнения — отметки о том, что привычка выполнена в конкретный день

Каждый пользователь видит и редактирует только свои данные (RLS).

Дизайн-вдохновение

Вдохновляемся дизайном и UX с сайта: https://uneevo.com/

Обрати внимание на стиль карточек, типографику, общий лейаут и плавные анимации.

Не пиши код. Сначала создай подробную документацию проекта: архитектура, структура базы данных в Supabase, ключевые фичи, структура файлов, и план реализации по шагам. Сохрани в файл PROJECT.md.

Также создай файл SETUP.md — инструкцию для разработчика: какие ключи и переменные окружения нужно получить и где, какие пароли/токены настроить (Supabase, Vercel и т.д.), и что нужно сделать вручную перед запуском проекта.
